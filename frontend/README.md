# ERP Frontend

## 1. Project Setup
Prerequisites: Node.js (v16+) installed.

Since the automatic installation might have been interrupted:
1. Open a terminal in `frontend` folder.
2. Run `npm install`.

## 2. API Proxy
The project is configured to proxy API requests to `http://localhost:9090`.
Make sure your backend Spring Boot server is running on port 9090.

## 3. Run Development Server
```bash
npm run dev
```

## 4. Build for Production
```bash
npm run build
```
